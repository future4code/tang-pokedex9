#### O que funciona:
- Adição e remoção de itens na pokedex com as devidas interações com a lista da home
- Responsividade nas telas de home, pokedex e detalhes
- Função de batalhar com pokemons aleatórios e chance de adiciona-los a sua pokedex caso ganhe a batalha

#### O que não funciona: 
- Responsividade na tela de batalha
- O desafio proposto de criar a comparação de pokemons não foi feito, optamos por criar uma feat de batalha
- Desafio de paginação não foi feito para que a mecanica de batalhas oferece a recompensa de acessar pokemons não mostrados na home
- Existe um bug legal q resolvemos não tirar, na pagina de detalhes do pokemon usamos o useParams para puxar os dados da API, como a API aceita tanto o ID (q é um número sequencial) como o nome do pokemon é possivel acessar os detalhes de qualquer pokemon q o usuário queira a partir dessa tela.
